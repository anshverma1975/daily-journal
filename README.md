# ◈ journal

*the minimal companion*

---

Most journaling apps try to do too much. This one doesn't.
Pick a date. Write. Save. That's the whole thing.

No tags. No folders. No markdown toolbar. No premium tier.
Just a calendar, a blank page, and whatever's on your mind.

---

## features

- **one page per day** — always has been, always will be
- **google sign-in** — no passwords to forget
- **dots on the calendar** — small, satisfying proof you showed up
- **future dates locked** — you write about the past, not the future
- **cmd/ctrl + s** — save like a developer, journal like a human
- **it remembers everything** — cloud-backed, nothing ever disappears
- **works on your phone** — for when thoughts strike at 2am

---

## stack

Next.js · NextAuth · Upstash Redis · Vercel

## run it yourself

```bash
git clone https://github.com/anshverma1975/daily-journal
cd daily-journal
npm install && cp .env.example .env.local
# add your keys to .env.local
npm run dev
```
